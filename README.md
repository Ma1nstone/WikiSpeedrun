# SpeedWiki

SpeedWiki is a fast-paced Wikipedia racing game where players compete to navigate from one Wikipedia article to another using only the links found within articles.

## How It Works

1. Start on a randomly selected Wikipedia article.
2. Receive a target article.
3. Navigate through Wikipedia using only article links.
4. Reach the target article in the fewest clicks and shortest time possible.
5. Compete against friends in real-time multiplayer races.

## Features

* Real-time multiplayer gameplay
* Custom race rooms with shareable codes
* Live leaderboard and player tracking
* Click counting and race timing
* Random article generation
* Responsive design for desktop and mobile
* Educational and knowledge-based gameplay

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/speedwiki.git
cd speedwiki
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm start
```

For development:

```bash
node server.js
```

The application will be available at:

```text
http://localhost:3000
```

## Gameplay Rules

* Only links within Wikipedia articles may be used.
* External links are not allowed.
* The winner is the first player to reach the target article.
* Players can compare click counts and completion times after each race.
