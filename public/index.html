How to Play

  1. Create or join a lobby with friends using a room code.
  2. The host starts the game. Everyone gets the same start & target articles.
  3. Click only Wikipedia links to navigate from the start article to the target.
  4. The first player to reach the target article wins instantly.
  5. There's no time limit — race as long as it takes!
  6. ⚠️ Using the Wikipedia search bar gives you a 30-second Chinese punishment!
