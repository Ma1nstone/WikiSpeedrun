Overview

Computer science is the study of computation, information, and automation. It involves both theoretical disciplines such as algorithms, theory of computation, and information theory, and applied disciplines such as the design and implementation of hardware and software.

The internet is a global system of interconnected computer networks that uses the Internet protocol suite (TCP/IP) to communicate between networks and devices. It is a network of networks that consists of private, public, academic, business, and government networks, linked by a broad array of electronic, wireless, and optical networking technologies.

Hardware

Computer hardware includes the physical parts of a computer such as the central processing unit (CPU), random access memory (RAM), storage devices, and input/output peripherals. Modern hardware design balances performance, power consumption, and cost.

  • CPU — executes instructions; measured in GHz clock speed and number of cores
  • RAM — temporary, fast-access memory used by running programs
  • Storage — HDDs and SSDs for persistent data; SSDs are significantly faster
  • GPU — graphics processing; increasingly used for parallel computation and AI
  • Motherboard — connects all components via buses and interfaces

Software

Software is a set of computer programs, instructions, and associated documentation. It can be divided into system software, which manages computer resources, and application software, which performs specific tasks for users.

Operating Systems

An operating system (OS) manages hardware resources and provides services for application programs. Major families include Microsoft Windows, macOS, Linux distributions, and mobile platforms such as Android and iOS.

Applications

Application software serves end-user purposes: word processors, browsers, databases, media players, and productivity suites. Open-source software allows users to inspect, modify, and distribute the source code under various licences.

Networking & the Internet

Computer networking involves transmitting data between nodes using wired or wireless technologies. The OSI model provides a conceptual framework of seven layers — from physical transmission up to application-level protocols like HTTP and SMTP.

Key concepts include IP addressing, DNS resolution, packet switching, and bandwidth management. The World Wide Web, built on top of the internet, uses HTTP/HTTPS protocols and HTML documents.

Security

Cybersecurity protects systems, networks, and programs from digital attacks. Common threats include malware, phishing, ransomware, and denial-of-service attacks. Defence strategies include firewalls, encryption, multi-factor authentication, and regular patching.

Programming

Programming is the process of creating software by writing instructions in a programming language. Paradigms include procedural, object-oriented, functional, and declarative programming. Widely used languages include Python, JavaScript, Java, C, C++, and Rust.

Web development specifically involves front-end technologies (HTML, CSS, JavaScript) and back-end server technologies (Node.js, Django, Rails, etc.), along with database management using SQL or NoSQL systems.
